¿Cómo funciona?

Este servidor HTTP está construido con Node.js y sirve archivos estáticos desde la carpeta public.

Los archivos se sirven según su tipo MIME, permitiendo que el navegador los interprete correctamente.

El servidor maneja errores comunes como archivos no encontrados (404) y errores internos del servidor (500).

Archivos disponibles

Puedes acceder a los siguientes archivos estáticos:

  • archivo.txt
  • data.json
  • imagen.png
  • style.css
  • script.js
